# 6styles Visual Pack

Six pluggable AI visual style templates: Traditional Art, Tech Particles, Collage, Liquid Color Flow, Tech Glass, Retro Newspaper.

## Styles

| # | Style | Category |
|---|-------|----------|
| 1 | New Year Print | Traditional |
| 2 | Tech Particles | Tech |
| 3 | Collage Vibe | Lifestyle |
| 4 | Liquid Color Flow | Lifestyle |
| 5 | Tech Glass | Brand |
| 6 | Retro Newspaper | Creative |

## Usage

1. Open the style JSON file
2. Copy the `formula` field
3. Replace `【】` variables with your content
4. Run the prompt in your image generation tool

## Rules

- Formula is fixed — only replace `【】` variables
- One theme at a time
- Title & font description on separate lines
- Color is an independent variable
