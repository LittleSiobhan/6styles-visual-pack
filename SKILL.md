# 🎨 六风格视觉包

六种跨场景 AI 生图风格，覆盖国风、科技、生活、品牌、文创五大品类。

## 风格一览

| # | 风格 | 线 | 一句话 |
|---|------|-----|--------|
| 1 | 🏮 非遗年画 | 国风 | 木版年画质感，套色印制 |
| 2 | ⚡ 科技粒子 | 科技 | 粒子流光+激光上升，发布会大屏 |
| 3 | 🏙️ 活力拼贴风 | 生活/活动 | 真人户外，涂鸦箭头，配色可换 |
| 4 | 🌸 液态色彩流 | 生活/活动 | 春日诗意，液态色彩交融 |
| 5 | 💎 科技玻璃风 | 品牌/商务 | 大面积留白，科技玻璃质感 |
| 6 | 📰 复古报纸风 | 创意/文化 | 活版印刷，文化专栏 |

## 使用方式

每套风格都是一个 JSON 文件，包含：
- `formula` — 固定 Prompt 公式，【】内为可替换变量
- `variables` — 变量说明与示例
- `canvas` — 推荐画面比例与分辨率

**使用步骤：**

1. 选择风格 → 打开对应 JSON
2. 复制 `formula` 字段
3. 替换【】内的变量为你需要的内容
4. **将完整 Prompt 提交给用户确认** → 用户审批通过后方可进入下一步
5. 将确认后的 Prompt 输入生图工具中生成

## 规则

- **公式固定不变**：除【】内的变量外，不要增加或修改任何描述
- **一次替换一个主题**：一套公式对应一套视觉语言，不要混搭
- **标题与字体分两段写**：标题文字单独一行，字体要求单独一行
- **颜色单独指定**：字体颜色作为独立变量，不靠"上文暗示"
