# 📋 六风格视觉包 · SOP

> 版本：v1 | 更新：2026-05-15

---

## 一、概述

本 SOP 适用于「六风格视觉包」的使用流程。该技能包包含 6 套即用视觉风格，采用「固定公式 + 可替换变量」架构，替换【】内内容即可快速出图。

## 二、角色说明

| 角色 | 职责 |
|------|------|
| **用户** | 选择风格、填写资料、确认 Prompt、选择生图工具 |
| **AI** | 展示风格、整理用户填写的资料、替换变量、调用生图工具出图 |

## 三、完整工作流

### Step 1：AI 展示 6 种风格供用户浏览

AI 将 6 种风格一次性展示给用户，包括名称、类别、一句话描述和适用场景。

```
🎨 六风格视觉包，请选择：

1｜🏮 非遗年画（国风）— 木版年画质感，套色印制
   适用：节气节日、传统文化、国潮营销

2｜⚡ 科技粒子（科技）— 粒子流光+激光上升，发布会大屏
   适用：发布会、科技品牌、大屏背景

3｜🏙️ 活力拼贴风（生活/活动）— 真人户外，涂鸦箭头，配色可换
   适用：户外活动、运动赛事、市集社区

4｜🌸 液态色彩流（生活/活动）— 春日诗意，液态色彩交融
   适用：春日主题、清新品牌、市集视觉

5｜💎 科技玻璃风（品牌/商务）— 大面积留白，科技玻璃质感
   适用：科技品牌、金融科技、创新大会

6｜📰 复古报纸风（创意/文化）— 活版印刷，文化专栏
   适用：文化专题、书评影评、怀旧主题

你想用哪个风格？请回复编号（1-6）。
```

### Step 2：用户选择风格

用户回复选择的风格编号或名称。

### Step 3：AI 列出需填写的变量，让用户填写

AI 打开对应风格的 JSON 文件，将 `variables` 字段中的变量整理成问题清单，请用户逐一回答。

**示例（以 2 科技粒子为例）：**
```
你选择了「科技粒子」，请填写以下信息：

1. 场景类型（如：产品发布会背景 / 品牌大屏背景 / 开场序幕）
2. 主色调（如：深蓝渐变 / 黑金渐变 / 科技蓝紫）
3. 粒子效果（如：光线粒子 / 数据流光 / 全息粒子）
4. 主标题文字内容
5. 年份
6. 品牌水印文字

请逐项填写 👇
```

### Step 4：AI 替换变量，与用户确认

用户填写完毕后，AI 将变量填入 formula 生成完整 Prompt，提交给用户审阅。

**确认文案：**
```
已根据你的填写生成完整 Prompt，请确认：

【完整 Prompt 内容】

确认无误吗？可以生图吗？
```

> 用户未确认前，绝对不可生成图片。

### Step 5：AI 询问用户使用什么生图工具

用户确认 Prompt 后，AI 询问用户根据实际情况选择生图工具。

**推荐文案：**
```
Prompt 已确认。你用什么生图工具？
```

### Step 6：出图给用户

用户指定工具后，AI 调用该工具生成图片，将结果发送给用户。如用户自行出图则无需 AI 介入。

---

## 四、规则红线

🚫 **不得跳过 Step 4（确认 Prompt）**——无确认不出图
🚫 **不得修改 formula 公式本身**——只替换【】内的变量
🚫 **不得混搭风格**——每套公式对应一套视觉语言
🚫 **不得增加额外描述**——公式外不增一词
🚫 **不得替用户选工具**——必须由用户决定

✅ **标题与字体分两段写**
✅ **颜色作为独立变量**
✅ **一条龙：展示 → 选择 → 填资料 → 确认 → 选工具 → 出图**
